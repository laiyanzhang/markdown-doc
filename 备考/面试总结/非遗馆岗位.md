# 非遗馆岗位
## 1.非遗馆内容
- 非遗项目：石湾陶塑、广东剪纸、佛山彩灯、英德红茶、流沙珍珠、三乡木偶、舞狮
- 非遗小记者：非遗直播间与非遗传承人对话，馆内学习英歌舞并拍摄短视频。集招募、探馆、研学、采访、直播与成果展示于一体的非遗教育品牌。项目以“非遗+教育”深度融合为路径，引领青少年从“走近非遗”迈向“走进非遗”，助力非遗从静态保护转向活态传承
- 青少年活动传播成果展：“青蓝韶华”展区，带你回顾馆内那些充满活力的场景，从“非遗种子计划”引领青年走进乡土非遗，到“志愿童行”夏令营开启跨馆美感之旅；从“童趣时光”亲子活动中的代际欢笑，到“我在非遗馆过寒暑假”系列中的探馆体验。记录着青少年从参观者到参与者、再到传播者的角色转变
- “广东文化遗产保护与利用”学术征文：主题聚焦“广东非遗传承的科技赋能与数字创意”，旨在回应数字时代非遗发展的新课题，推动传统文化在现代语境中焕发新生，期待广东非遗在科技赋能的加持下，更好地服务社会经济大局
- 广东“节庆‘叹’非遗”品牌：节假日通过进景区、举办墟市、体验课、分享会、研学课堂、生活工坊、展览、展演等多元形式，让非物质文化遗产更贴近百姓生活、让更多人民群众了解中华优秀传统文化
- “非遗向未来”系列展览：联合各地市非物质文化遗产保护中心系统梳理全省非遗资源，充分激活地域联动、资源共享的协同发展机制，促进各地市之间的文化交流。展览中引入AI互动装置、观众留言共创等互动形式，构建“观看-体验-反馈”的完整闭环，营造可感、可参与、可对话的活态现场
- 智慧化场景建设：联动高校、科技企业、创新品牌等，香云纱艺术装置采用VR交互技术、3D打印拼装潮玩、3D建模广彩联名潮玩、AI里“活起来”的石湾陶塑作品，联通的《AI英歌舞》通过VR眼镜实现前端交互，让英歌舞IP“活”起来；枫溪瓷烧制技艺文创作品《福蝶》通过AR增强现实技术，实现实物与移动端的交互体验；优创合影的《傩舞走清将》通过AR眼镜和一体机AI成像处理技术让观众可以定制属于自己的傩舞影像；《广府梦华录》和《广州非遗街区（北京路）》通过数字孪生打造元宇宙空间，实现沉浸式体验，让观众穿梭到古代岭南的繁华街市中，沉浸式感受广州作为古代海上丝绸之路重要发祥地的历史记忆
- 非遗传播形式
  - 集市摊位：“喜番造物社”邀请非遗创新品牌主理人经营非遗潮玩摊位，打造传统文化潮玩站；节日摊位集市，推动内需，刺激消费，售卖非遗好物；“冬季非遗主题系列活动-滋补人生”非遗好物展销区售卖中药、茶叶，助力打造具有岭南特色、可持续、可推广的文化消费新场景，为广东高质量发展注入持久而鲜活的文化动能
  - 闯关游戏：“我在非遗馆过寒假”亲子非遗文化闯关游戏，解谜心得分享至小红书、微博等社交平台
  - 分享会：开展读书分享会分享非遗文化、品茶会分享茶艺文化
  - 动手制作：传承人手把手教学+沉浸式体验+创意DIY制作二天油香牌、裁剪剪纸、制作彩灯
  - 表演：“非遗+体育”结合全运会中非遗相关元素，引入机器狗醒狮，展开体育主题活动。节日非遗节目文艺汇演蔡李佛拳、醒狮、龙川杂技等
  - 市场经济：高埗矮仔肠制作技艺从一杆矮仔肠到一个现代化食品企业的蜕变历程，生动诠释了如何通过标准化生产、品质把控与品牌营销，将传统技艺转化为具有市场竞争力的商品，让学员们看到了非遗项目实现自我“造血”的可行性与广阔前景；佛山木版年画通过融入潮流元素、开发文创产品等方式走进年轻群体，揭示了非遗的生命力在于与时代脉搏同频共振，在于与当代人、特别是年轻人的情感共鸣与价值共享
  - 非遗游戏：《咏春行》将咏春拳化作像素风格的叙事探索RPG游戏，以“非遗+游戏”的形式为未来岭南武术传播构建了深度体验、互动操作的全新探索；《叫我大掌柜》将广府建筑、洪拳招式、醒狮跳桩等非遗元素融入游戏




## 2.网络安全
- 金句：网络安全是智慧化场馆的生命线。非遗馆存储了大量珍贵的非遗数字资源，一旦泄露或丢失，损失不可估量。我的工作重点将围绕等保2.0合规建设展开，通过技术防御（如防火墙、WAF）和管理制度（如应急预案、人员培训），构建‘防得住、查得清、恢复快’的安全体系，确保文化数据的安全
- 技术防御：部署防火墙、WAF（Web应用防火墙）严格隔离办公网与业务网，并对非遗核心数据实施加密存储。
- 管理运维：建立资产台账和漏洞扫描机制，定期进行安全巡检，杜绝弱口令。
- 应急响应：制定《网络安全事件应急预案》，明确断网、数据泄露等场景的处置流程，确保在突发情况下能快速恢复业务。”
- 网络瘫痪处理
  - 先处置：第一时间断网隔离，启用备用线路或离线预约方案，优先恢复业务（让观众能进馆）
  - 再上报：同步向馆领导汇报情况，说明影响范围及初步原因
  - 后排查：联合技术团队排查原因（是DDoS攻击还是设备故障），进行溯源分析
  - 终复盘：事件结束后，形成书面报告，优化防御策略，避免再次发生
- 保障非遗数字资源的安全
  - 管理层面：建立资产台账，明确数据责任人，定期进行安全意识培训
  - 技术层面：对核心非遗数据实施加密存储，部署访问控制策略，防止越权访问
  - 运维层面：建立漏洞扫描和日志审计机制，确保所有操作可追溯
- 网络安全“三要素”
  - 保密性（C）：防止非遗数据泄露
  - 完整性（I）：防止数字展品被篡改
  - 可用性（A）：确保预约系统、官网7x24小时可访问
- 等保2.0：各位考官，如果由我负责落实非遗馆的等级保护工作，我将严格遵循“定级、备案、建设、测评、运维”五步法，重点抓好以下三个环节：第一，定级备案是前提。​ 我会牵头组织专家，将非遗馆的核心系统（如预约系统、数字资源库）科学定级为二级或三级，并按规定向公安机关备案。这一步的关键是摸清家底，明确哪些数据是必须重点保护的“核心资产”。第二，对标建设是关键。​ 依据等保2.0标准，重点构建“一个中心、三重防护”体系。具体来说，就是通过部署防火墙、WAF等技术手段，结合访问控制、日志审计等管理制度，确保非遗数据的保密性、完整性和可用性。第三，持续运维是保障。​ 等保不是“一次性考试”。我会建立定期的漏洞扫描、应急演练和人员培训机制，确保系统持续合规，能够应对新型网络威胁，真正守护好我们的非遗数字资源。总之，我的工作思路是：以合规为底线，以数据安全为核心，用体系化的管理为智慧场馆保驾护航。”


## 3.智慧化场景建设
- 金句：非遗馆的智慧化，核心是用技术（AR/VR/AI）解决传统展览‘看不懂、留不住、体验浅’的痛点。它不是简单的设备堆砌，而是通过数字化手段，让非遗从‘静态展示’变为‘可互动、可感知、可传承’的沉浸式体验。例如，通过AR技术让观众‘试穿’传统服饰，利用AI算法推荐个性化非遗路线。最终目标是降低理解门槛，增强文化认同
- 基础设施层：构建高速、稳定的全馆网络覆盖，这是所有智慧应用的‘地基’。
- 数据资源层：建立非遗数字资源库，对传承人技艺、口述史进行数字化采集与存储，这是非遗馆区别于普通场馆的核心。
- 应用体验层：部署智能导览、预约系统，并利用AR/VR技术打造沉浸式体验（如我馆已有的‘AI讲古人’项目），让非遗‘活起来’。
- 智慧化项目主要面临三类风险
  - 技术风险：设备兼容性差、网络延迟导致体验卡顿
  - 体验风险：内容同质化，或过于追求炫技，脱离了非遗的文化内核
  - 数据风险：观众在使用互动设备时产生的个人信息，需做好隐私保护，防止数据泄露
- 案例结合
  - AR/VR应用：如“粤剧脸谱AR互动”、“潮汕木雕VR虚拟拼接”等具体场景
  - AI应用：如利用AI进行“非遗知识问答”、“观众行为分析以优化展线”
  - 合作模式：强调“馆校合作”中，高校负责技术研发，馆方负责内容把关的产学研模式
