# 复习资料
## 一、常识
- 政治理论：选项内容需要紧扣题干关键词
- 法律问题：一般不可对抗善意第三人
- 一般对
  - 对人民好（人＞企业＞钱）
  - 说中国好
  - 有领导人
  - 重复要素较多的选项
  - 根据直觉很容易做出判断为错实际正确的选项，比如某个事情具备特殊的规则，看起来似乎不合理实则却是正确
- 一般错
  - 出现精确数字
  - 选项中存在分类
  - 过于绝对的说法
  - “既可以...又可以...”的选项一般错误
  - “张冠李戴、偷换主体”


## 二、言语理解
- 中心理解
  - 主体：正确选项一般包含题干中提及的主体，常见坑点在于主体范围扩大
  - 举例：举例内容一般不作为核心概括；判断举例是否为重点，若举例为正面案例则非重点，若举例提出亟需论点内容则重点
  - 位置：根据选项内容匹配文段位置，若位于理由、影响、描述说明等位置时则非重点；位于特别是、尤其是等后面为重点
  - 对策：一般实干比空谈更正确。有针对性对策优选针对性对策，无针对性对策或对策错误，选择问题本身的同义替换
  - 转折词：但是、因此、如果、一旦等，转折词前的内容一般不作为中心句
- 词句理解：通过指示代词前一句作判断，只需关注主谓宾而非状语
- 关键词提炼：关键词包括主体、重点（问题+对策）
- 语句填空：横线在中间：承上启下，承启的均为观点而并非例子
- 接语选择
  - 要点：关注文段最后一句话，找出最后论述的话题，接下来的论述也要与核心话题相关
  - 常见干扰项：文段中已经论述过的内容（前面提到部分相关内容，后续仍可展开叙述）
  - 常见顺序1：是什么 -> 为什么 -> 怎么做（常考）
  - 常见顺序2：当前技术/研究 -> 新技术/研究
- 逻辑填空
  - 语境铺垫：“有恃无恐”需前面提及对应“恃”、“独树一帜”需前面提及其他主体
  - 两并列：前后词语近义或反义，但词性必须一致
  - 三并列：三者均为同一方面或均为不同方面
  - 递进：语义倾向一致，但程度前轻后重
  - 对比择优：程度轻重、感情色彩、比喻拟人
- 细节判断
  - 偷换概念：A比B更、强加因果、因果倒置
  - 偷换时态：过去/现在/将来
  - 偷换语气：慎选绝对化表达，优选温和表达（重点判断一定、可能等）
- 语句排序
  - 确定首句：适合-下定义、背景引入、提出观点；不适合-关联词后半部分、指代词、“比如”
  - 确定捆绑：指代词捆绑对应名词、关联词捆绑（虽然但是、不仅而且）
  - 确定顺序：时间顺序、逻辑顺序（先A后B）、关联词顺序（一方面+另一方面）
  - 确定尾句：多为结论或对策
  - 常见顺序：先写利用A来做什么事，再到这样做的意义
- 篇章阅读
  - 策略：快速标段落序号、通过问题判断做题顺序、根据问题锁定文段阅读
  - 做题顺序：通读理解全文类型题目后置
  - 语句对应原文位置：联系前一句匹配关系、优先匹配文字少的段落
  - 技巧：题目顺序与文段顺序基本一致


## 三、资料分析
### 1.挖坑点
- 构成：出现‘其中’、‘包括’等词语时需判断主体的结构构成
- 多多少倍/是多少倍
- 增长量/增长率
- 推出/不能推出

### 2.技巧
- 答案分布：5道题中一般同时包含ABCD四个正确答案，最后1题可根据前4题缺少的选项进行优先判断或者直接选择
- 第5题：综合判断题一般比较耗时可斟酌放弃，判断选项顺序可按照CDAB的顺序判断
- 最大值/最小值：直接判断四个选项之间的大小
- 乘法应用：选项差距较小时，可直接估算一个值做乘法计算。如根据现期以及增长率计算基期时，可大致估算一个基期乘以增长率判断是否接近现期
- A=B*C中计算A的增长率：给出B与C的增长率时应用间隔增长率


### 3.速算
- 乘数一大一小：按比例增加或减小，如 9 * 11 ≈ 10 * 10
- 多数相加：小数相加则先算整数部分相加；整数相加则先算首位数相加（适用于选项相差小）
- 分数：$\frac{1}{6}$ ≈ 16.7%、$\frac{1}{7}$ ≈ 14.3%、$\frac{1}{8}$ ≈ 12.5%、$\frac{1}{9}$ ≈ 11.1%、$\frac{1}{13}$ ≈ 7.7%、$\frac{1}{14}$ ≈ 7.1%、$\frac{1}{15}$ ≈ 6.7%、$\frac{1}{16}$ ≈ 6.25%
- 分数比较：直除首位、变化速度


### 4.题型策略
- 前提：今年A的增长率a，B的增长率b
- 变化幅度：增长率的绝对值
- 间隔增长率：r = $r_1$ + $r_2$ + $r_1$$r_2$（$r_1$$r_2$ < 1%时可忽略）
- 混合增长率：量与距离（与混合增长率的差距）呈反比
- 年均增长值：增长值/年数
- 年均增长率：增长率/年数得平均增长率，年均增长率必小于等于平均增长率
- 累计增速与当前增速：若当前的累计增速相比前面上升，则当前增速 > 累计增速
- 当前正/负增长：将历年以来的增长率相加，和数为负数则必定为负增长
- 总价增长率：根据单位数量增长率以及总量增长率通过间隔增长率公式计算
- 平均增长速度：无法通过增长速度的算术平均数计算得出，还需得知各部分总量
- 对比增量：增速相近时，现期越大，增量越大
- 比重变化：部分增长率 > 整体增长率，部分的比重上升
- 基期比重/基期倍数：$\frac{A/(1+a)}{B/(1+b)}$ = $\frac{A}{B}$ * (1 + b - a)（前提是 a = ±20% 或选项差距大）
- 现期基期比重差：$\frac{A}{B}$ * $\frac{a-b}{1+a}$
- 平均值变化：$\frac{a-b}{1+b}$



## 四、判断推理
### 1.类比推理（国考）
- 核心方法：结合两个词语造句，加字越少越好，加字尽量加相同字
- 近义/反义
- 比喻象征
- 逻辑关系
  - 全同关系
  - 并列关系：矛盾、反对、相同/相似功能、连续排列、动力来源、命名方式
  - 包容关系：种属、组成、特指
  - 交叉关系
  - 对应关系：材料、工艺、功能、属性、时间顺序、依据、因果、职业、方式目的、配套使用、必要条件、地点
- 语法关系：主谓关系、动宾关系、主宾关系、偏正关系（修饰与被修饰）
- 二级辨析：感情色彩、程度、词性、形容对象
- 拆分思维：将1个词语拆分为2个词语或2个单字，优先拆分为词语，拆分后可能应用于所有关系


### 2.逻辑判断
- 翻译推理：只要A就B = A → B ； 只有A才B（除非A否则不B） = B → A；A → B = -B → -A
- 组合推理：通过多个对象之间关联的信息推理
  - 题干信息明确且选项信息充分：从题干明确信息入手，逐个排除选项
  - 题干出现“可能/不可能”、“补充哪个条件”：从选项入手，逐个代入题干排除或从重复出现最多的信息入手
  - 蒙题：正确选项选题干中出现次数最少的
- 真假推理：题干中的论述有真有假且真假未知
  - 将 A → B 转化为 -A 或 B
  - 先找出矛盾/反对关系，从而确定其他条件的真假
  - 根据其他条件的真假再去反推矛盾/反对关系中的真假
  - 若无矛盾/反对关系，使用假设法寻找条件中相同/相反的判断条件进行假设
- 加强
  - 力度区别：搭桥 > 必要条件 > 解释原因 > 举例
  - 二级辨析：明确 > 不明确；整体 > 部分；必然 > 可能
- 削弱
  - 力度区别角度1：削弱论点 > 拆桥 > 削弱论据
  - 力度区别角度2：因果倒置 > 另有他因
  - 不是A而是B导致某结果：优先找A导致B的选项
  - A有效/有害是因为A的提取物有效/有害：A的提取物极少/需要摄入大量A
  - 由部分推出整体：部分不能代表整体/其他部分没有相同规律
  - 主体：优先寻找主体一致选项，不存在举与主体不一致的反例去削弱的情况
- 原因解释：不降反升/不升反降题型，选项内容要包含反升/反降的原因



## 五、图形推理
### 1.规律
- 位置规律
  - 平移：直线（上下、左右、斜对角线）、绕圈（顺、逆时针）、平移导致区块重叠、到达边框后调转方向继续平移
  - 旋转：顺时针、逆时针，常见度数：45/90/180
  - 翻转：竖轴对称左右翻转、横轴对称上下翻转
- 样式规律
  - 相加/相减
  - 去异求同
  - 去同求异
- 属性规律
  - 对称性：对称轴的方向规律变化、数量相等/递增、经过的点线面特性；中心对称
  - 曲直性：全曲、全直、有曲有直
  - 开闭性：全封闭（半开半闭有时算封闭）、全开放
- 数量规律
  - 点数量：交点、切点、曲直交点
  - 线数量：直线、曲线、平行线、数量差、首末线垂直/平行、首末线方向相同/相反、一笔画、笔画数
  - 面数量：三角形的面数量、四边形的面数量、立体图形的外表面数量、最大/最小面的形状、属性、面积、边数、相似性、对称性、边框相似
  - 素数量：元素种类、元素个数、元素换算、元素位置（相邻/相对）、一定范围内出现元素种类与个数相同（九宫格内一行/一列）
  - 角数量：直角数
- 标记元素
  - 点：直直交点/曲曲交点/曲直交点/由点发射线条数量
  - 线：直线/曲线/最长边/最短边/对称轴/线中点
  - 角：锐角/直角/钝角/最大角/最小角
  - 面：重合面/单独面/最大面/最小面/相同形状面
  - 双标记特殊：连线与所在图中其他图形的关系
  - 箭头指向：箭头指向性、箭头与箭头间垂直/平行
- 图形间关系
  - 相离、相压/重合、相交
  - 相交于点
  - 相交于面：相交面的形状、面积、属性
  - 相交于边：相交边的数量、属性（最长边/最短边/包含/交错/曲直）

### 2.策略
- 九宫格：先横看，再竖看，最后S形看（甚至米字型看）
- 拆分思维
  - 位置规律：区分内圈与外圈的平移/旋转规律
  - 样式规律：外框线求同、内框线求异
  - 数量规律：区分框内与框外的点/线数量
- 规律叠加：存在同时满足两种规律的情况
- 常见数量关系：相等、等差、加减相等、加减等差
- 六面体折叠
  - 相对面：同行/同列相隔一个面、紧邻Z字形两端的面
  - 相邻面：优先判断有方向性的面的相邻面
  - 公共边：平面上构成直角的两条边、4个面连成行/列时分处两端的两条边
  - 公共点：对比平面图中的公共点以及立体图中的公共点，图形方向是否一致
  - 画边法：寻找一个面中具备方向性的点，在立体图与平面图中同时顺时针描边，对比平面图与立体图中是否一致
- 非常规立体折叠
  - 四面体：展开图中同一条直线上的两条边是公共边；以当前面作箭头，判断箭头左侧面以及右侧面
  - 八面体：每个点同时在4个面上，构成外角为钝角的两条边折合后是公共边；以当前面作箭头，判断箭头左侧面以及右侧面
  - 常规技巧：底面边数 = 侧面面数；公共边等长且成对出现
  - 其他：针对展开图中有区别的平面的边，寻找邻边长度相等
- 立体拼合
  - 方块数量不同：优先通过数量排除选项
  - 方块数量相同：从方块数最多/形状与待拼成的立体图形相似的图形入手
  - 用于拼合的立体图形均成对：拼合后的立体图形对称
  - 蒙题：较复杂立体拼合直接蒙A或B
- 截面图
  - 六面体：矩形、梯形、锐角三角形、正六边形
  - 圆柱：圆、椭圆、矩形，侧切两边都是曲线
  - 圆锥：圆、椭圆、三角形
  - 圆台：圆、椭圆、梯形
  - 内部挖空：截面图对应部分出现空缺
  - 正方体内挖空圆锥/圆台：斜切时椭圆不能在截面图正中间
  - 注意：不存在的实线
- 三视图：外部轮廓定角度，内部线条判对错；遮挡部分/挖空部分不画线或画虚线
- 黑白块
  - 平移（形成内外圈时需根据内圈数量是否一致判断是否应用拆分思维）
  - 黑白运算（线段、阴影、空白均可运算）
  - 笔画数：连接形成粗线条
  - 对称性：整体/黑白分开看对称性
  - 黑白部分数：相等、加减等差
  - 加减同异
- 形似S、Z、N、平行四边形、风车：中心对称
- 明显存在2条垂直对称轴：轴对称 + 中心对称
- 等腰图形：轴对称
- 多个对称图形组成：对称轴平行/垂直、对称轴 + 中心对称
- 五角星、形似“日”、形似“田”、穿串：一笔画
- 汉字：笔画数、面数量、线数量
- 线条交叉明显/相切较多、外部轮廓相似内部线条变化：点数量
- 多边形、单一直线：直线数量
- 曲线图形：曲线数量
- 平行较多：平行线对数
- 单条线折来折去：首末线关系
- 图形被分割、封闭面明显、封闭的数字/英文/汉字：面数量
- 完整图形留下开口：封闭性、角数量
- 明显横平竖直特征、类似电话卡等被改造的图形：直角数量
- 每幅图只存在2个元素并且元素个数差异无规律：元素换算
- 生活化、粗线条图形：开闭性、面数量、部分数
- 线条穿过白球宫格：划分白球个数、穿过白球个数
- 走投无路试试笔画


## 六、科学推理（省考）
### 1.物理
- 静摩擦力：物体静止时，大小随外力的变化为变化，最大值近似等于动摩擦力
- 动摩擦力：摩擦力方向与物体运动方向相反，等于压力乘以摩擦系数
- 物体静止或匀速运动：受力平衡，若无外力推动则摩擦力也为0
- 液体压强：ρgh
- 气体压强：体积反比、温度正比
- 浮力
  - 公式：ρgV~排~
  - 漂浮：浮力 > 重力
  - 悬浮：浮力 = 重力
  - 沉底：浮力 < 重力
- 受力分析：重力 + 弹力（支持力） + 摩擦力
- 匀速直线运动或静止状态判断弹力/摩擦力是否存在：对已知力受力分析，若存在某个方向的力无法抵消，则存在可分解到对应方向的弹力/摩擦力
- 合力与两分力夹角相等时，合力不变时随着夹角减小，两分力均逐渐减小
- 向心力：m$\frac{v^2}{r}$
- 滑轮组：省力情况看动滑轮有几段绳子，有n段则只需$\frac{1}{n}$重力的拉力
- 杠杆：阻碍杠杆转动的力为阻力，让杠杆转动的力为动力
- 平面滑轮组：依旧能发挥省力的作用
- 绕过滑轮的同一根绳子上的力的大小处处相等
- 杠杆：动力垂直于杠杆时最省力
- 惯性：仅和质量有关，质量越大，惯性越大
- 加速度：F=ma，加速度方向与合外力方向一致
- 匀速圆周运动：合外力提供向心力，向心力大小不变
- 经过拱桥：做圆周运动，整体合外力指向圆心
- 人在车内运动：物体与人车分离时，人车需要当做一个整体去看合外力
- 速度-时间图像：覆盖面积代表位移
- 位移-时间图像：斜率代表速度
- 速率：根据路程计算而并非位移，速率没有正负
- 路程：位移有正负，路程没有正负
- 透明物体：绿色物体只允许绿色光透过
- 光的折射：垂直水面作法线，光线与法线夹角，空气一侧夹角大于水下一侧夹角；岸高水浅，偏折为实际，直线为假象
- 透镜：凹透镜发散，凸透镜汇聚
- 音调：敲击装水容器，水柱越长音调越低；向装水容器吹气，空气柱越长音调越低
- 光的色散：红光折射率最小，紫光折射率最大
- 水面倒影：倒影与实物关于水面对称
- 做功：W=Fs，力与位移在同一方向做正功；不在同一方向做负功
- 机械能：只有重力或弹力做功的系统内，机械能守恒（需注意是否存在摩擦力做功）
  - 动能：$\frac{1}{2}$mv^2^
  - 重力势能：mgh
  - 弹性势能：形变量越大，弹性势能越大
- 沸腾：需要达到沸点且持续吸热
- 热传递：由高温物体传递到低温物体、温差越大传递越快
- 电荷：被丝绸摩擦过的玻璃棒带正电；被毛皮摩擦过的橡胶棒带负电
- 电荷相互作用：同性相斥、异性相吸，带电物体可以吸引不带电的轻小物体
- 电压表：与被测用电器并联，可视为断路
- 串联电路：
  - 电流处处相等
  - 总电阻等于各串联电阻之和
  - 总电压等于各串联电压之和，分压比等于电阻比
- 并联电路：
  - 各支路两端电压相等
  - 总电阻的倒数等于各并联电阻倒数之和（并联用电器电阻越大，总电阻越大；并联用电器越多，总电阻越小）
  - 总电流等于各并联电路电流之和
- 开关：连接在火线和用电器之间
- 双孔插座：左零右火
- 三孔插座：零线地线均无电
- 短路：并联电路中，若某电路中不存在用电器，则会使另外电路短路
- 电源电压：用电器直接连在电源两端，用电器电压等于电源电压

## 七、数字推理（省考）
- 图形数阵
  - 角度：上下、左右、交叉
  - 法则：加减乘除、倍（周围数字之和的倍数）、方（周围数字之和的平方）、最小公倍数、最大公约数
- 表格数阵：行规律，列规律、斜线规律
- 数字位数均很多且变化起伏不定：优先考虑机械拆分，选项中寻找同类型数字
- 根号：整数平方并入根号 / 根号中化简出整数
- 数字增大呈现为位数级变化，出现极大数：优先考虑幂次方，判断数字是否与平方立方的数字相近
- 均为小数：小数点前后拆分来看前后两部分是否各自有规律，既要将单个数字作为一个数分析，又要将两个数字作为一个数分析
- 基础数列：等比、等差、质数、合数
- 全为分数或大部分为分数：分数常常搭配其他数列进行考察，通分与反约分仅为转化为其他数列的手段
- 项数较多，增大减小不规律：优先考虑多重数列，隔项/分组
- 递推数列：前面数列变化平缓，后面数列变化巨大；圈三个数作和方积倍，大数易有规律
- 多级数列：无明显规律，需要作差作商作和作积

### 2.选项分析

- 半径：选项同时存在半径、直径
- 半天内次数：选项同时存在半天内与一天内次数

### 3.题型
1. 整除/余数：一个数能被3/9整除，那么其各位数之和能被3/9整除
2. 倍数、百分数、分数、比例：需要满足对应的倍数
3. 300多/70多：根据比例倍数特性，得出准确数字
4. ax^2^+bx+c求最大值：-$\frac{b}{2a}$
5. ax^2^+bx+c=0求最大值：两解x~1~和x~2~的平均值
6. 溶液混合、平均数混合、利润率混合、折扣混合、比重混合、增长率混合：线段法（混合增长率）
7. 存在三量关系A=B×C且只给出一个量的具体值：对另外两个量中的一个进行赋值
8. 均是倍数、分数、百分数、比例，无实数：赋值公倍数或者简单数1/2/10/60/100等
9. 溶液问题，溶质不变，溶剂变化：定值份数-定单值
10. 经济利润问题，总价不变，比例变化：定值份数-定和值
11. 年龄问题，年龄差不变，年龄变化：定值份数-定差值
12. 等总价平均单价、等溶质平均浓度、等工作量平均效率：调和平均数 $\frac{2×C~1×C~2}{C~1+C~2}$
13. 人、物品等确定为非负整数，方程解存在多个解：奇偶、倍数、尾数方法逐一列举可能解
14. 只有两条三元一次方等式却要求出x+y+z：假设x为0后化简为两条二元一次方程求解
15. A、B、C三支队伍，期间单干或合作，不停歇地完成工作：三支队伍的工作时间 = 总工程量 / 三支队伍的总效率
16. 甲单独30天完成，乙单独45天完成：设置最大公倍数90，则甲效率为3，乙效率为2，两者合作90/5=16天完成
17. 甲与乙的效率比为2:3，合作8天完成：总工程量2×8+3×8=40，甲单独完成40/2=20天
18. A、B队共同施工，B队期间停工n天，C队后面加入，最后依旧按时完成：C队工作用于填补B队停工的n天，可得出B队与C队的效率之比
19. A单干x天，A、B合作y天：A与B效率之比 y ：x-y
20. A效率比B效率大a，A花x天完成，B花y天完成：B花 (y-x) 天完成 ax 的工作量
21. 初始效率5x，中途效率下降80%，导致延长y小时完工：(5x-x) × z = xy，得出低效率的工作时长
22. 上下坡/往返/前后半程：等距离平均速度V~平均~=2V~1~V~2~/(V~1~+V~2~)
23. 流水行船：V~顺~=V~船~-V~水~；V~逆~=V~船~+V~水~
24. 相遇追及：T~相遇~=S/(V~1~+V~2~)；T~追及~=S/(V~1~-V~2~)
25. 环形跑道：追及代表多跑一圈；相遇代表合跑一圈
26. 两端往返跑相遇（迎面）：每n次相遇，两者合跑2n-1次全程
27. A与B速度之比6:5，中途A加速x，经过y小时与甲同时到达：xy用于弥补1份速度的差值，6xy为全路程
28. 同向同时同点追及，A最快、B中等、C最慢：假设C不动，A追及C的时间即为A跑一圈的时间
29. 同向同时同点追及，A快，B慢，每次追及后A减速，直至两者速度相同：列出相遇的时间节点，数据相加
30. 两者相遇与追及所需时间的比例x/y：列式：x(a-b) = y(a+b)，得出a与b的速度之比
31. 两端往返跑相遇问题，给出A与B第x与y次相遇时的路程：总路程在第x与y次相遇时比值为2x-1:2y-1，同时A/B单人行程在第x与y次相遇时比值也为2x-1:2y-1
32. 两人相向相遇问题，给出相遇所需时间x以及相遇后继续走完所需时间y：可得出速度之比x：y
33. 环形跑道三个同时同点出发，速度各不相同，求解相聚起点的时间：速度之比 = 路程之比 = 圈数之比，相聚时间 = 任一人跑的圈数 × 其单圈所需时间
34. 路程包含上下坡以及平路且时间只有总时间：全程平均速度 = 上下坡平均速度 = 平路
35. 必须相邻：相邻的捆绑为一个单位后再进行排列，排列完后捆绑的内部再进行排列
36. 必须分隔：其余先排列，分隔的单位在排列中插空
37. 至少n个：a人分b个物品，每人至少n个，a人先各自拿n-1个，剩余d个物品；在d-1个位置选a-1个位置放隔板
38. 除自己对应位置外其他位置均可：错位排列D~4~=9，D~5~=44
39. 圆桌就座：排列方式有(n-1)!种
40. 部分按顺序，部分不能在某位置，部分需相邻：先将按顺序的部分排序，再将需相邻的捆绑后插空再进行内部排序，最后将不能在某位置的进行插空
41. 概率问题中涉及两个区间交叉概率问题：绘制函数图像，x、y轴各为一个区间的取值范围，标记关键节点并连线，形成的阴影面积占据x、y轴形成的正方形面积的比例即为概率
42. 计算两人在同一组的概率：先计算A在某个组中的概率，再计算B也选中该组的概率，两者相乘为在某个组的概率，将所有组的概率分类计算后相加即为两人在同一组的概率
43. 计算两人相邻/同一排的概率：A随便坐，只需要计算B在剩余座位恰好选中符合条件的座位即可
44. 至少...保证：最不利情况 + 1
45. 总量一定，排名第n的人最多或最少：数列构造，构造数列-求谁设谁-反推其他-加和求解
46. 分为多个集合，至少多少人满足所有集合特征：多集合反向构造，数量 = 总量 - 不满足各集合特征人数之和
47. 两集合容斥：A + B - A∩B = 总数 - 都不
48. 满足两项型三集合容斥：A + B + C - A∩B - A∩C - B∩C + A∩B∩C = 总数 - 都不
49. 只满足两项型三集合容斥：A + B + C - 只满足两项 - 2×满足三项 = 总数 - 都不
50. 只满足一项型三集合容斥：画图法，圈内标记只满足数，圈上标记满足数
51. 构造相似三角形：面积的比例 = (边长的比例)^2^
52. 四边形蝴蝶定理：四边形被两条对角线切割为四个三角形，对角三角形面积相乘相等（P137-12）
53. 圆O~1~绕圆O~2~无滑动滚动一圈回到出发点：圆O~1~半径为r，圆O~2~半径为R，R=kr，则圆O~1~自转(k+1)圈
54. 线段同侧存在A、B两点，求线段某点与两点距离之和最短：A点以线段作为对称轴作镜像A1点，A1点与B点连线，连线与线段交点为所求点，即距离之和最短的点

56. 圆锥体积之比：相似比的立方
57. 同等周长矩形最大面积：形状越接近正方形面积越大
58. 棱长为n的立方体涂黑，切割为若干个棱长为1的小立方体：未被涂黑的小立方体(n-2)^3^，1面涂黑6(n-2)^2^，2面涂黑12(n-2)，3面涂黑8个
