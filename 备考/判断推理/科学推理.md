# 科学推理
## 一、物理
### 1.摩擦力
- 静摩擦力：物体静止时，大小随外力的变化为变化，最大值近似等于动摩擦力
- 动摩擦力：摩擦力方向与物体运动方向相反，等于压力乘以摩擦系数
- 物体静止或匀速运动：受力平衡，若无外力推动则摩擦力也为0

### 2.压强
- 封闭气体压强
  - 温度不变，压强与体积成反比
  - 体积不变，压强与温度成正比
- 液体压强 = ρgh

### 3.浮力
- 浮力 = ρgV~排~
- 浮沉条件
  - 漂浮：浮力 > 重力
  - 悬浮：浮力 = 重力
  - 沉底：浮力 < 重力


### 4.受力分析
- 匀速直线运动或静止：合外力为0
- 受力分析：重力 + 弹力（支持力） + 摩擦力
- 匀速直线运动或静止状态判断弹力/摩擦力是否存在：对已知力受力分析，若存在某个方向的力无法抵消，则存在可分解到对应方向的弹力/摩擦力
- 弹力：必定垂直于接触面
- 相互作用力：大小相等、方向相反
- 斜面对物体的作用力：支持力 + 摩擦力
- 斜面对物体受力分析常规思路：物体重力正交分解分别对应支持力与摩擦力
- 合力与两分力夹角相等时，合力不变时随着夹角减小，两分力均逐渐减小
- 向心力：m$\frac{v^2}{r}$


### 5.简单机械
- 滑轮组：省力情况看动滑轮有几段绳子，有n段则只需$\frac{1}{n}$重力的拉力
- 杠杆：阻碍杠杆转动的力为阻力，让杠杆转动的力为动力
- 平面滑轮组：依旧能发挥省力的作用
- 绕过滑轮的同一根绳子上的力的大小处处相等
- 杠杆：动力垂直于杠杆时最省力


### 6.运动定律
- 惯性：仅和质量有关，质量越大，惯性越大
- 加速度：F=ma，加速度方向与合外力方向一致
- 匀速圆周运动：合外力提供向心力，向心力大小不变
- 经过拱桥：做圆周运动，整体合外力指向圆心
- 人在车内运动：物体与人车分离时，人车需要当做一个整体去看合外力


### 7.运动图像
- 速度-时间图像：覆盖面积代表位移
- 位移-时间图像：斜率代表速度
- 速率：根据路程计算而并非位移，速率没有正负
- 路程：位移有正负，路程没有正负


### 8.光
- 透明物体：绿色物体只允许绿色光透过
- 光的折射，垂直水面作法线，光线与法线夹角，空气一侧夹角大于水下一侧夹角；岸高水浅，偏折为实际，直线为假象
- 透镜：凹透镜发散，凸透镜会聚
- 音调：敲击装水容器，水柱越长音调越低；向装水容器吹气，空气柱越长音调越低
- 光的色散：红光折射率最小，紫光折射率最大
- 水面倒影：倒影与实物关于水面对称


### 9.功和能
- 做功：W=Fs，力与位移在同一方向做正功；不在同一方向做负功
- 机械能：只有重力或弹力做功的系统内，机械能守恒（需注意是否存在摩擦力做功）
  - 动能：$\frac{1}{2}$mv^2^
  - 重力势能：mgh
  - 弹性势能：形变量越大，弹性势能越大
- 沸腾：需要达到沸点且持续吸热
- 热传递：由高温物体传递到低温物体、温差越大传递越快


### 10.电
- 电荷：被丝绸摩擦过的玻璃棒带正电；被毛皮摩擦过的橡胶棒带负电
- 电荷相互作用：同性相斥、异性相吸，带电物体可以吸引不带电的轻小物体
- 电压表：与被测用电器并联，可视为断路
- 串联电路：
  - 电流处处相等
  - 总电阻等于各串联电阻之和
  - 总电压等于各串联电压之和，分压比等于电阻比
- 并联电路：
  - 各支路两端电压相等
  - 总电阻的倒数等于各并联电阻倒数之和（并联用电器电阻越大，总电阻越大；并联用电器越多，总电阻越小）
  - 总电流等于各并联电路电流之和
- 开关：连接在火线和用电器之间
- 双孔插座：左零右火
- 三孔插座：零线地线均无电
- 短路：并联电路中，若某电路中不存在用电器，则会使另外电路短路
- 电源电压：用电器直接连在电源两端，用电器电压等于电源电压


## 二、化学
- 氧气：不能燃烧，但有助燃性
- 金属活动性
  - 含义：和氧气、盐酸、稀硫酸反应的活性，氢左侧金属可以和盐酸、稀硫酸反应
  - 由强到弱：钾钙钠镁铝锌贴锡铅（氢）铜汞银铂金
- 铝：导电性、延展性，生成致密氧化铝薄膜，保护里层不再反应
- 汽油：溶解油污
- 酸碱指示剂
  - 紫色石蕊溶液：酸红碱蓝
  - 无色酚酞溶液：酸无碱红
- 苏打水：碱性
- 臭氧：杀菌消毒利用的是强氧化性
- 碳
  - 还原性：用于冶金工业
  - 金刚石：最硬
  - 石墨：导电
  - 活性炭：吸附性
- 二氧化碳：能溶于水，与水反应生成碳酸，密度比空气大（沉到底层）
- 一氧化碳：难溶于水，火焰呈蓝色，有还原性，密度比空气小
- 还原性：有失去电子、化合价升高的趋势（电子为负价）
- 氧化性：有得到电子、化合价降低的趋势
- 溶液配置
  - 配置方法：溶质左物右码天平称量，溶剂量筒称量，烧杯混合
  - 溶液浓度偏大：俯视量筒、天平指针偏右、砝码生锈、水溅出
  - 溶液浓度偏小：仰视量筒、天平指针偏左、砝码缺角、左码右物（使用游码）
- 无色气体鉴别
  - 氧气：火星木条复燃，燃着的木条烧得更旺
  - 二氧化碳：澄清石灰水变浑浊
  - 一氧化碳：点燃产生气体使澄清石灰水变浑浊，燃烧颜色为蓝色
  - 氢气：点燃后烧杯内壁出现水珠，燃烧颜色为蓝色
  - 甲烷：点燃后烧杯内壁出现水珠，且生成能使澄清石灰水变浑浊的气体，燃烧颜色为蓝色
  - 其他气体：氮气无色、二氧化硫刺激性气味且能让品红褪色、氨气是碱性气体
- 硬水软水：肥皂水。有浮渣产生，无泡沫或泡沫较少是硬水；无浮渣，泡沫较多是软水


## 三、生物
- 生态系统
  - 能量最终来源：阳光
  - 生物部分：生产者、消费者、分解者（可以没有消费者）
  - 食物链：生产者、消费者
  - 种群关系：共生-荣辱与共、捕食-此消彼长、竞争-你死我活
  - 生物富集：单向流动、逐级递减，营养级越高生物数量越少
- 细胞
  - 动植物均有：细胞膜、细胞核、细胞质、线粒体
  - 植物有：细胞壁、液泡、叶绿体
  - 病毒：无细胞结构
- 有丝分裂：
  - 间期：DNA复制以及蛋白质合成，占据大部分时间
  - 前期：出现纺锤丝
  - 后期：染色体翻倍
- 染色体：
  - 数量：22对常染色体、1对性染色体
  - 成分：DNA和蛋白质
- 基因：
  - 显性基因/显性遗传：AA、Aa
  - 隐性基因/隐性遗传：aa
- 血型：
  - A型：I^A^I^A^/I^A^i
  - B型：I^B^I^B^/I^B^i
  - AB型：I^A^I^B^（万能受血型）
  - O型：ii（万能输血型）
- 光合作用
  - 反应：二氧化碳 + 水 + 光能 -> 有机物 + 氧气
  - 影响因素：加强光照、升高二氧化碳浓度、温度适宜，加强光合作用（有上限）
  - 注意：植物生根发芽后才可进行光合作用，此前均只进行呼吸作用消耗
- 呼吸作用
  - 有氧呼吸：有机物 + 氧气 -> 二氧化碳 + 水 + 大量能量
  - 无氧呼吸（植物/酵母菌）：产酒精
  - 无氧呼吸（动物/乳酸局）：产乳酸
  - 影响因素：升高二氧化碳浓度越高、降低温度，抑制呼吸作用
- 酶
  - 定义：活细胞产生有催化作用的有机物
  - 成分：绝大多数为蛋白质，部分RNA
  - 高效性：远胜一般催化剂
  - 专一性：只能催化一种或一类反应
  - 温度影响：高温失活无法恢复，低温降低活性可恢复
  - PH：过高或过低都会使酶彻底失活


## 四、地理
- 时区：东十二区衔接西十二区，地方时换算东加西减
- 经线：20度西经以及160度东经区分东西半球，本初子午线0度，往东往西经度增大
- 等高线地形图
  - 坡度判断：密陡疏缓
  - 山脊：等高线向低凸出
  - 山谷：等高线向高凸出
  - 鞍部：山峰之间较平缓地方，适合露营
- 六大板块：亚欧、非洲、美洲、印度洋、太平洋、南极洲
- 降水
  - 海陆位置：距海近，降水多
  - 山脉走向：横向山脉阻挡水汽进入内陆
  - 地形因素：迎风坡降水多
- 风：
  - 风向：风吹来的方向
  - 风形成：高气压流向低气压（会受到地转偏向力的影响）
  - 风速：等压线越密集、风速越大
- 地球自转：自西向东，北极看逆时针、南极看顺时针，太阳、月亮、星星都东升西落
- 地转偏向力：南半球物体朝运动方向左侧偏转，北半球物体朝运动方向右侧偏转
- 热力环流：绿化带建设在环流内，化工厂建设在环流外
